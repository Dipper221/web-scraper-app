# Web Content Scraper (Client-Server)

Приложение для поиска, скачивания и сохранения веб-контента.
Реализовано на Node.js (Express) + Vanilla JS.

## Возможности
1. Поиск по ключевым словам (зашиты в базе сервера: `js`, `text`, `html`).
2. Проксирование запросов через сервер для обхода CORS.
3. Отображение прогресса скачивания (размер, проценты).
4. Сохранение контента в LocalStorage браузера.
5. Просмотр сохраненного контента оффлайн.
